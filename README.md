# 项目架构 
* 采用前后端分离，JS + SpringBoot2.0    
* 后端语言Java  
* ORM使用SpringDataJPA    
* 数据库使用MySql5.7+Mongodb3   
* 缓存Redis  
* 部署Nginx+Jar
